## ✨ Animations You'll See

### **On Page Load (Entrance):**
1. ⏱️ **0.0s** - Background orbs fade in
2. ⏱️ **0.2s** - Social icons appear (staggered)
3. ⏱️ **0.4s** - Badge slides in
4. ⏱️ **0.6s** - Name fades up
5. ⏱️ **0.8s** - Typing animation starts
6. ⏱️ **1.0s** - Description appears
7. ⏱️ **1.2s** - Quote box slides in
8. ⏱️ **1.4s** - Buttons appear
9. ⏱️ **1.6s** - Stats fade in
10. ⏱️ **1.0s** - Profile picture zooms in
11. ⏱️ **1.4s** - Signature fades in

### **Continuous (Always Running):**
1. 🔄 **Gradient flow** - Name colors shift (3s loop)
2. 🔄 **Floating profile** - Up/down motion (4s loop)
3. 🔄 **Pulsing badge** - Green dot pulses (infinite)
4. 🔄 **Background orbs** - Soft breathing (infinite)
5. 🔄 **Lightning rotation** - Spins 360° (20s loop)
6. 🔄 **Rocket bounce** - Up/down (3s loop)
7. 🔄 **Typing effect** - Cycles through roles (2s each)

### **On Hover:**
1. 🎯 **Social icons** - Scale 1.15x + shift right
2. 🎯 **Buttons** - Lift up 2px + scale 1.05x
3. 🎯 **Profile** - Scale 1.05x + rotate 2°
4. 🎯 **Primary button** - Gradient shifts Purple→Pink
