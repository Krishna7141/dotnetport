## ?? Getting Started

### Prerequisites
- Node.js 16+ and npm/yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Krishna7141/dotnetport.git
   cd dotnetport
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## ?? Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run ESLint
```

## ?? Customization

### Update Personal Information
- Edit `src/Components/HeroSection.jsx` for bio and social links
- Modify `src/Components/WorkSection.jsx` for work experience
- Update `src/Components/ContactSection.jsx` for contact details

### Change Theme Colors
- Edit `tailwind.config.js` to customize color palette
- Modify `src/index.css` for global styles

### Add/Remove Sections
- Update `src/App.jsx` to add or remove sections
- Create new components in `src/Components/`
