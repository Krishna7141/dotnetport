## ?? CSS Animations Added

### **New in `index.css`:**
```css
@keyframes gradient          - Flowing gradient text
@keyframes glow-border       - Pulsing border glow
@keyframes gradient-shift    - Shifting button gradients
@keyframes neon-glow         - Neon text effect
```

### **Utility Classes:**
```css
.animate-gradient           - 3s gradient animation
.animate-gradient-shift     - Shifting gradient background
.glow-border               - Animated glowing border
```
